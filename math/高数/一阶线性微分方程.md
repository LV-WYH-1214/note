一阶线性齐次微分方程 #card

**标准形式**：$y' + P(x)y = 0$
^1773189018524

**通解公式**：

$$y = C e^{-\int P(x) \, dx}$$

**推导思路**：它是最基础的类型，可以直接通过分离变量法 $\frac{dy}{y} = -P(x)dx$ 两边积分推导出来。

一阶线性非齐次微分方程 #card

**标准形式**：$y' + P(x)y = Q(x)$ （注意 $y'$ 的系数必须化为 $1$）
^1773189018528

**通解公式**：

$$y = e^{-\int P(x) \, dx} \left( \int Q(x) e^{\int P(x) \, dx} \, dx + C \right)$$

**结构解析**：通解 = 对应的齐次方程通解 + 非齐次方程的一个特解。公式里的 $e^{\int P(x) \, dx}$ 被称为**积分因子**。
