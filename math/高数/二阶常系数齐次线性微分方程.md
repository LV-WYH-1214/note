二阶常系数齐次线性微分方程 $y'' + py' + qy = 0$ 的特征方程 #card

$r^2 + pr + q = 0$


特征方程有两个不相等实数根 ($r_1 \neq r_2$) 时的通解公式 #card

$y = C_1 e^{r_1 x} + C_2 e^{r_2 x}$


特征方程有两个相等实数根 ($r_1 = r_2 = r$) 时的通解公式 #card

$y = (C_1 + C_2 x) e^{rx}$


特征方程有一对共轭复数根 ($r_{1,2} = \alpha \pm i\beta$) 时的通解公式 #card

$y = e^{\alpha x} (C_1 \cos(\beta x) + C_2 \sin(\beta x))$


二阶常系数非齐次线性微分方程解的结构 #card

$y = Y + y^*$ (即：对应齐次方程的通解 $Y$ + 该非齐次方程的一个特解 $y^*$)
